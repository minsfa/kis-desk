KIS 자동매매 상태 보고서 (2026-06-04)

코드 위치: ~/kis-autotrade. 원칙: 기본 dry-run, 검증된 것만 라이브, 1주 단위, 한도/킬스위치/전수로깅.

== 1. 계좌/연결 현황 ==
- 계좌: 한국투자증권 위탁 (실전, 계좌번호는 config/.env)
- 잔고: 원화 약 1,600,138원 + 외화 약 327 USD
- 국내(KRX/NXT) + 미국 둘 다 실거래 체결 검증 완료 = M1(배관) 완성

== 2. 실거래 검증 결과 ==
(미국) 일반주식 T(AT&T) 1주 매수->체결->매도->청산 성공. SCHD 등 ETF는 "해외ETP 거래 미신청"으로 거부 -> MTS에서 해외ETP 거래 신청해야 ETF/레버리지 가능.
(국내 NXT 프리마켓 08:00~08:50) 우리금융지주 1주 지정가 매수(30,050)->매도(29,750)->청산 성공.
  - 거래소 라우팅 필드 EXCG_ID_DVSN_CD="SOR"(스마트라우팅)로 NXT 프리마켓 체결 확정.
  - 교훈: 호가단위(tick) 준수 필수. 2~5만원대=50원 단위. 코드에 자동 라운딩 반영함.
(반복매매) T 1/2/3주 왕복 3회 연속 성공. 주당 왕복비용 약 0.27달러(=공격적 스프레드 횡단 기준, 타이트 지정가면 훨씬 낮음).

== 3. 통합 PoC 도구 (최종 정리) ==
명령: python -m src.cli pocall [--session ...] [--live]
현재 KST 시각으로 세션을 자동 감지해 시장/라우팅/주문유형/호가단위를 맞춰 사고팜.
- nxt_pre   08:00~08:50  국내 NXT 프리   SOR  지정가(+tick)
- kr_main   09:00~15:30  국내 메인(KRX)  KRX  시장가
- nxt_after 16:00~20:00  국내 NXT 애프터 SOR  지정가(+tick)
- us_reg    22:30~05:00  미국 정규장     해외 지정가
- closed    그 외        dry 미리보기(체결 불가)
종목: 국내 ETF 360750(TIGER 미국S&P500), 국내 주식 316140(우리금융지주), 미국 주식 T. (전부 비레버리지)
체결확인은 잔고(balance) 기반(체결조회 ccnl 불안정 회피). 라이브는 .env DRY_RUN=false일 때만.

== 4. 주가 로깅 테스트 ==
N종목 x 간격 폴링 -> CSV(data/pricelog_YYYY-MM-DD.csv) 적재 + 처리량/지연 측정.
결과(프리마켓): 1종목0.5s, 5종목1s, 10종목0.5s 모두 무에러. 지연 약 67~82ms, 실효 약 14 쿼리/s(한도 20/s 아래, 자동 throttle). 시세 시계열 축적 토대 확보.

== 5. OpenClaw 연계 ==
OpenClaw는 오퍼레이터(명령 실행+보고)만. 주문 결정은 코드가, DRY_RUN 전환은 사람만.
cron으로 pocall을 시각 맞춰 실행->텔레그램 보고하는 흐름 검증됨(어제 US, 오늘 국내).
예: 09:01 cron kis-pocall-main -> pocall --live (DRY_RUN=true면 dry).

== 6. 안전장치 ==
기본 dry-run, 라이브 게이트(DRY_RUN=false), 1회 주문한도(국내 5만원/해외 100달러), 일일매매횟수 한도, STOP 파일 킬스위치, 지정가 호가단위 자동 라운딩, 전 주문 JSONL 로깅, fail-closed.
