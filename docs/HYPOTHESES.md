## A. 시장 레짐

- 상태: ✅ **데이터 확정.**
- 함의: 대형주 신규 장기진입 = 천장 매수 위험(비대칭). **신규 장기는 안 오른 가치/laggard에서.**

**A2. 코스닥은 laggard라 catch-up 여지가 있다.**
- 근거: 코스닥150 200일선 +11.5%, 6개월 +27%(KOSPI +143% 대비 크게 뒤짐). 최근 1~3개월은 약세.
- 상태: ⚠️ **부분.** "안 올랐다"는 확정, "이제 오른다"는 미확정 → 관찰.

## B. 수급

## C. 매매 전략
**C1. (눌림목+매수벽) 우상향 대장주가 순간 급락 + 호가 매수벽 받침일 때 받아서 반등에 판다.**
- 대상: 반도체 대장주·레버리지 ETF·코스닥 대장주(하락추세 제외). 선별적(매일 X), 기다렸다 지정가.
- 조건: flash dip 도달 + **호가 매수/매도 잔량비 높음(매수벽)** [+ 외인 수급 가점].
- 근거: 호가 잔량비 실시간 작동 확인(삼성 105배 매수벽 vs 하이닉스 0.5 매도우위). 레버리지 변동성 6.2%/일.
- 상태: 🔧 **데이터 준비완료, 검증 진행**(페이퍼테스트 + 분봉). v0 대상.

**C2. (전날 급등→익일 눌림) 전날 +5~8% 급등 종목은 다음날 눌렸다 반등한다.**
- 근거(5년, 1,245건): 다음날 시가→저가 -2.7%, 저가→종가 **+2.45%(승률 91%)**. 현실판(시가-2% 지정가→종가청산) 추정 +1.7%/건.
- 상태: 🟡 **데이터 지지, 단 분봉 경로·비용·코스닥 적용 검증 필요.** (일봉은 고저 순서 모름)
- 활용: 코스닥 성장주 급등 모니터링 → 익일 눌림 매수 후보.
